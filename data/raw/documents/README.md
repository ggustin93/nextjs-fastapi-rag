# Source Documents

Place documents here for ingestion into the knowledge base.

**Supported formats:** PDF, DOCX, DOC, PPTX, XLSX, HTML, MD, TXT

**Structure:**
```
documents/
├── BusinessAnalysis/   # Business & functional analysis
├── Slides/             # Training presentations
└── UserGuides/         # User documentation
```

```bash
make ingest  # Process all documents in data/raw/
```
